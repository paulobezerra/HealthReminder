Privacy Policy - Health Reminder

Last Updated: May 17, 2025

1. Introduction

The "Health Reminder" extension was developed to help users maintain healthy habits by providing reminders to stand up, drink water, and use the bathroom while using the computer. This Privacy Policy explains how we handle your data.

2. Data Collected

The extension collects the following data:

  • Local Settings: The reminder interval (in minutes) set by the user is stored locally on the device using the `chrome.storage` API. This data is not sent to external servers.
  • Anonymous Analytics Data: We use Google Analytics 4 (GA4) to collect anonymous usage data, such as events for installation, popup openings, and notification displays. This data does not include personally identifiable information.

3. Use of Data

The data is used to:

  • Persist your reminder interval settings across sessions.
  • Monitor general usage of the extension (via Google Analytics) to improve the user experience and fix issues.

4. Data Sharing

We do not share personal data with third parties. Analytics data sent to Google Analytics is anonymous and used solely for statistical purposes.

5. Security

Data stored locally (reminder interval) is securely saved on the user's device and is not accessed by third parties.

6. Changes to This Policy

We may update this policy from time to time. The latest version will always be available on this page.
