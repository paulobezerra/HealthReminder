Política de Privacidade - Lembrete de Saúde

Última atualização: 17 de maio de 2025

1. Introdução

A extensão "Lembrete de Saúde" foi desenvolvida para ajudar os usuários a manterem hábitos saudáveis, exibindo lembretes para levantar, tomar água e usar o banheiro. Esta Política de Privacidade explica como lidamos com seus dados.

2. Dados Coletados

A extensão coleta os seguintes dados:

  • Configurações locais: O intervalo de tempo (em minutos) definido pelo usuário para os lembretes é salvo localmente no dispositivo usando a API `chrome.storage`. Esses dados não são enviados para servidores externos.
  • Dados analíticos anônimos: Usamos o Google Analytics 4 (GA4) para coletar dados anônimos sobre o uso da extensão, como eventos de instalação, abertura do popup e exibição de notificações. Esses dados não incluem informações pessoais identificáveis.

3. Uso dos Dados

Os dados são usados para:

  • Manter suas configurações de intervalo entre sessões.
  • Monitorar o uso geral da extensão (via Google Analytics) para melhorar a experiência e corrigir problemas.

4. Compartilhamento de Dados

Não compartilhamos dados pessoais com terceiros. Os dados analíticos enviados ao Google Analytics são anônimos e usados apenas para fins estatísticos.

5. Segurança

Os dados salvos localmente (intervalo de tempo) são armazenados de forma segura no dispositivo do usuário e não são acessados por terceiros.

6. Alterações nesta Política

Podemos atualizar esta política ocasionalmente. A versão mais recente estará sempre disponível nesta página.
